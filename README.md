# Among-Us-3D
Proyecto académico, realizado en Godot y Blender, enfocado en la creación e integración de entornos y personajes 3D.

## Descripción

El objetivo del proyecto fue diseñar modelos 3D en Blender e integrarlos en Godot para crear una experiencia interactiva.

## Tecnologías utilizadas

- Godot Engine
- Blender

## Características

- Modelos 3D propios
- Integración de assets 3D
- Trabajo en equipo
